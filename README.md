## Що це

**Alliance Pro** — розширення для Chrome/Edge, яке додає плаваючу панель на сторінку
корпоративного чату `chat.sender.ftband.net`. Воно допомагає оператору вести розмову
за чек-листом, формувати готовий текст для копіювання, відстежувати час і робити нотатки —
усе унікально для кожного чату. Працює **окремо** від Sender Pro і не конфліктує з ним.

## Можливості

| Вкладка | Опис |
|---|---|
| **Чек-лист** | Настрій клієнта (😠/😐/😊/🔄), вибрані пункти → `Зроблено`, невибрані → `Залишилось`. Пункти можуть мати незалежні під-варіанти (`Назва(А, Б)`). Поле «Комент» — окрема частина тексту. Текст копіювання оформлено смайликами. Кнопки «Скопіювати» / «Скинути», живий перегляд, перетягування пунктів, профілі чек-листів. |
| **Таймер** | Хвилини/секунди з «позичанням» хвилини, звуковий сигнал (вибір звуку), а також **Будильник** на конкретний час із кнопками +15/+30/+1год. Таймер **глобальний і фоновий** — працює, навіть коли вкладка неактивна чи браузер згорнутий, і синхронізований між усіма сторінками (один таймер, а не окремий під кожен сайт). |
| **Калькулятор** | Миша + повна підтримка клавіатури (цифри, нумпад, `+ - * / = Enter Backspace Esc %`). |
| **Нотатки** | Окрема нотатка під кожен чат + блок «Корисні посилання» (налаштовувані кнопки-посилання). |

Додатково: світла/темна тема, вибір кольору акценту, **повзунок розміру панелі**,
компактний режим, адаптив під вузькі/малі екрани, «Відкрити РС» + авто-визначення
ID клієнта, автоочищення даних раз на добу, експорт/імпорт усіх налаштувань і
конфігурації чекбоксів (файлом або вставкою JSON), збереження стану окремо для
кожного чату.

**Калькулятор і таймер на інших сайтах:** у налаштуваннях можна задати список
сайтів, де показуватиметься міні-панель лише з вкладками «Калькулятор» і «Таймер»
(без чек-листа й нотаток). Тому розширення має доступ до всіх сайтів — але панель
зʼявляється тільки на доменах зі списку.

## Встановлення

1. Завантажте репозиторій: `Code → Download ZIP` (або `git clone`) і розпакуйте.
2. Відкрийте `chrome://extensions` (або `edge://extensions`).
3. Увімкніть **Режим розробника** (Developer mode) у правому верхньому куті.
4. Натисніть **Завантажити розпаковане** (Load unpacked) і виберіть папку репозиторію
   (ту, де лежить `manifest.json`).
5. Відкрийте `chat.sender.ftband.net` — панель зʼявиться у правому верхньому куті.

## Швидкий старт

- Відмічайте пункти чек-листа під час розмови → натисніть **Скопіювати** — готовий
  текст у буфері обміну.
- **Скинути** очищає поточний чат (можна ввімкнути авто-скидання після копіювання).
- ⚙️ — налаштування (тема, колір, звук, розмір, чекбокси, профілі, очищення).
- ⊟ — компактний режим (згортає панель по висоті).

Детальна інструкція — у [`docs/USAGE.md`](docs/USAGE.md).

## Структура

```
manifest.json        — маніфест розширення (MV3)
css/alliance.css     — стилі панелі (світла/темна тема, адаптив)
js/
  storage.js         — сховище (localStorage, профілі, налаштування)
  theme.js           — тема, акцент, масштаб
  draggable.js       — перетягування панелі
  operatordesk.js    — CID + «Відкрити РС»
  calculator.js      — калькулятор, таймер, будильник
  notes.js           — нотатки під чат
  checklist.js       — чек-лист і формування тексту
  settings.js        — налаштування та модальні вікна
  main.js            — побудова панелі, вкладки, перемикання чатів
icons/               — іконки розширення
docs/                — документація
```
